Txn-Viewer

1. Open chrome console
2. console: account.getTransactions()
3. console: account.balance()
4. console: account.addTransaction(100,'BP','1/1/2015','sample')
5. console: account.getTransactionsByTypes(['BP','IT'])
6. console: account.getTransactionsByDateRange('1/1/2015','2/1/2015')

Javascript purely logic based, no UI is implemented.

Assumptions
- the rest of the transactions require an amount, type, date and description.
- transaction type can only be these 3 types (BP, FT, IT)
- invalid transaction type will default to empty ''
- display amount will round down to 2 decimal places
- empty amount will default to 0
- empty or invalid date will default to empty ''
- display date format will be DD/MM/YY 01/01/2015
- no sorting implementation
- the app is for development and it not ready for production use
